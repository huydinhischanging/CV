# AI-assisted workflow notes

This folder documents how AI (Claude Code) was used while building this project, for reviewer transparency.

This project was built iteratively with Claude Code: the developer was taught React fundamentals (components, JSX, props, `useState`) step by step, wrote the Sidebar collapse logic themselves after guided exercises, and used AI assistance to scaffold the remaining sections (Projects, Interests, EndSection), fix responsive CSS issues, and set up the Netlify deployment configuration (`netlify.toml`, `netlify-cli` devDependency, and deploy instructions in `CLAUDE.md`/`README.md`).

No credentials, session transcripts, or tokens are stored in this repo — this folder only contains a summary of the collaboration, not raw AI chat logs.
